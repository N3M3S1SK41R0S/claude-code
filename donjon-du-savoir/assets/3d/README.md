# Collection 3D locale

Les modèles de ce dossier sont des créations originales pour **Le Donjon du
Savoir**. Ils utilisent des formes low-poly et des matériaux PBR en aplats de
couleur, sans texture externe.

- `heroes/` : 11 figurines avec quatre animations ;
- `buildings/` : 10 bâtiments-repères ;
- `tiles/` : 12 socles sculptés, un par illustration de case ;
- `dungeon/` : mur, arche, colonne et brasero réemployés comme modules.

Tous les GLB sont compressés avec `KHR_draco_mesh_compression`. Les PNG
historiques et les primitives Three.js restent les replis immédiats. Les
détails de poids et de géométrie sont dans `data/modeles-3d.json` et
`data/decors-3d.json`.
